# cmake_frame
cmake project frame

目录结构：

cmake_frame/				根目录
|----------/conf			配置文件目录，放置项目相关的配置文件
|----------/dbscript			数据库脚本目录。放置项目相关的数据库脚本
|----------/dep				依赖第三方目录，放置项目依赖第三方开源的源代码或者相关文件
|----------/doc				项目文档目录，放置项目相关文档
|----------/examples			测试代码目录，放置项目中模块测试、使用示例方法源代码
|----------/include			项目依赖的头文件目录，放置项目依赖的第三方开源源代码头文件放置目录，分文件夹存放
|----------/lib				项目依赖的库文件目录，放置项目依赖的第三方开源源代码编译成静态库、动态库，库放置目录
|----------/make_demo			工程编译示例目录，放置工程示例目录
|----------/shell			脚本目录，放置项目中shell、python脚本
|----------/src				源代码目录，放置项目中所有源代码
